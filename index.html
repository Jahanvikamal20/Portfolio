Chapter 03 — Proof of Work

Two problems, two builds

Reading about skills only tells you so much — here's where they actually got used.

PROJECT 01

TalentForge — AI Resume Analyzer

The problem: most resumes never make it past an ATS filter, and most people don't know why.
  • Built an AI-powered resume analyzer that evaluates ATS compatibility, generates ATS scores, and extracts relevant keywords from uploaded resumes.
  • Implemented AI-driven improvement suggestions and job-matching by comparing resumes directly against job descriptions.
  • Integrated a mock interview module that generates role-specific technical and HR interview questions to help users prepare.
React.jsNode.jsExpress.jsMongoDBTailwind CSSGoogle Gemini API
PROJECT 02

BookVault — Library Management System

The problem: tracking books, members, and returns by hand doesn't scale.
  • Developed a full-stack library management system using the MERN stack to manage books, users, and book issue/return operations.
  • Designed an interactive dashboard with real-time statistics — total books, issued books, returned books, and registered users — powered by RESTful APIs.
  • Implemented role-based authentication for Admin and User accounts, with full CRUD functionality for efficient book and user management.
React.jsNode.jsExpress.jsMongoDBTailwind CSS
